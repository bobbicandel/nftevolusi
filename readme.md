# Autonomous NFT Evolution Engine

Fully autonomous generative NFT ecosystem running **entirely on GitHub Actions**.

No servers  
No docker  
No API keys

## Features

- DNA inheritance
- Mutation engine
- Natural selection
- NFT metadata generation
- Trait system
- Rarity calculation
- PNG artwork generation
- GIF evolution timeline
- Auto README gallery

## Structure

art/ → generated artworks  
metadata/ → nft metadata  
stats/ → rarity + collection data  
gif/ → evolution animation

## Evolution

Every workflow run creates a new generation of organisms.
